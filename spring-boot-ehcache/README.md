The application is a simple cache manager on Spring Boot and EhCache.

#### Prerequisites:
- Java 8
- Maven 3

#### Technologies used:
- Spring-Boot 1.5
- Tomcat 8

#### Usage

Build and Run:
	
	mvn clean install
	mvn spring-boot:run

Preview at:

	http://localhost:8080/springboot/cache
