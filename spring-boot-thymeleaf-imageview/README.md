The application is a simple image viewer based on Spring Boot, Thymeleaf and Fancybox lightbox scripts.

#### Prerequisites:
- Java 8
- Maven 3

#### Technologies used:
- Spring 5
- Spring-Boot 2.2
- Tomcat 9
- Thymeleaf 3.0

#### Usage

Build and Run:
	
	mvn clean install
	mvn spring-boot:run

Preview at:

	http://localhost:8080/imageview
